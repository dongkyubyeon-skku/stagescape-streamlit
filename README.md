# StageScape

## K-pop Concert Technology Visual Archive

StageScape is a Streamlit-based web archive that organizes K-pop concert stage technologies, venue scale, production systems, and visual direction.

The project was created to explore concerts from a production perspective rather than a fan-oriented perspective.

---

## Project Goal

The goal of this project is to build a searchable and visual archive of K-pop concert production technologies.

StageScape helps users explore:

- which technologies are used in concerts
- how different concerts use similar technologies
- how venue scale affects stage production
- how visual direction supports audience experience

---

## Main Features

### 1. Technology → Concert Analysis

Users can select a stage technology and find concerts that use it.

Example:
LED Wall → BTS, BLACKPINK, SEVENTEEN, TWICE

---

### 2. Concert → Technology Analysis

Users can select a concert and see the technologies used in that concert.

Example:
BLACKPINK BORN PINK WORLD TOUR → LED Wall, Kinetic Stage, Laser System

---

### 3. Visual Reference Gallery

Instead of embedding unstable image files directly, the website provides visual reference links.

Users can move to external pages to check actual concert footage and stage visuals.

## Dataset

- Artist / Event

- Venue
- Scale
- Audience
- Year
- Production
- Technology Tags
- Production Analysis
- Visual Reference Link

---

## Research Method

The data was collected through publicly available sources such as:

- official concert videos
- official performance clips
- stage recordings
- concert trailers
- award show performances
- entertainment news articles
- production-related interviews

The concerts were analyzed based on visible stage elements and production structure.
