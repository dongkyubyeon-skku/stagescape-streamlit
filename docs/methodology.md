# Methodology

## Objective

To create a searchable archive of K-pop concert production technologies.

## Data Collection Process

1. Select representative K-pop concerts and events.
2. Review official concert footage and stage recordings.
3. Identify stage technologies used in each production.
4. Categorize technologies using a tag system.
5. Organize information into a searchable archive.

## Technology Categories

- LED Wall
- XR
- AR Camera
- Projection Mapping
- Kinetic Stage
- Automated Lift
- Moving Runway
- Laser System
- Drone Show
- Virtual Production

## Archive Perspective

The archive was developed from a concert production and stage design perspective.
